# Pasta de Imagens de Produtos

Alexsander Xavier - 4338139

## Estrutura

Esta pasta (`/public/assets/images/produtos/`) armazena as imagens dos produtos do cardápio.

## Como adicionar imagens

1. Adicione arquivos JPG/PNG com os nomes conforme lista abaixo
2. O frontend detectará automaticamente e exibirá
3. Se não houver imagem, usa placeholder

## Nomes dos arquivos esperados

### Pratos Executivos
- `baiao-de-dois.jpg`
- `carne-de-sol.jpg`
- `escondidinho.jpg`
- `galinha-caipira.jpg`
- `moqueca-siri.jpg`
- `acaraje.jpg`

## Resolução recomendada

- Mínimo: 300x300px
- Recomendado: 400x400px
- Máximo: 800x800px

## Formato

- JPG: Melhor compressão para fotos
- PNG: Para imagens com transparência

## Nota

O sistema foi implementado com fallback para não quebrar caso a imagem não exista.
